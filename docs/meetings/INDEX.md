## 前測共識重點

### 必做事項（會議結論交集）
1. **分級通報標準**：明確的升階觸發條件
2. **SBAR 交班內容**：人數、情緒、是否持械、環境狀況
3. **角色分工**：主責溝通、支援聯絡、環境控管
4. **低刺激通報**：無聲警報 + 暗語系統
5. **撤離動線**：單門/雙門標準策略
6. **跨單位一致性**：精神科與一般門診處置差異

### 持續改進項目
- 駐警裝備配置（精神科 vs 一般門診差距）
- 約束設備更新
- 緊急壓扣鈴無聲化
- 鎮定藥物處置流程統一
