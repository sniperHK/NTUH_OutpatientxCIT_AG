# M06 通報升級與法規

> **學習目標**：學員能辨識需要升級通報的四項指標，正確執行通報決策樹，並了解事件後續處理流程、人員關懷資源與相關法規。

---

## 一、升級條件四指標

當門診暴力事件出現以下**任一情形**，應立即從言語降階升級至通報處理：

| 指標 | 具體表現 | 威脅等級 |
|------|---------|---------|
| **言語威脅** | 「我要打你」「信不信我砸了這裡」「你給我小心」 | 中高 |
| **持物品揮舞** | 拿起椅子、點滴架、水壺、雨傘等物品 | 高 |
| **逼近醫護人員** | 跨越安全距離（< 1 公尺）、步步進逼、堵住出口 | 高 |
| **無法溝通** | 完全不回應、持續吼叫、疑似酒醉或精神狀態異常 | 中高 |

> **判斷原則**：不需要「四項全中」才升級。任何一項出現，即應啟動通報流程。寧可過度通報，不可延誤處置。

---

## 二、通報決策樹

根據事件性質，採取不同的通報路徑：

### 路徑 A：言語暴力

```
言語暴力（大聲叫罵、言語羞辱、威脅）
    │
    ├─ 護理師先以「停聽同選」規勸
    │
    ├─ 規勸無效 → 通知當班 Leader 到現場評估
    │
    ├─ Leader 評估後決定 → 聯繫駐警隊
    │
    └─ 駐警到場規勸 → Leader 評估是否需聯繫管區派出所
```

### 路徑 B：肢體暴力

```
肢體暴力（推擠、拍打、丟擲物品、肢體攻擊）
    │
    ├─ 立即聯繫駐警隊
    │
    ├─ 駐警到場 → 立即主動聯絡管區派出所到院
    │
    └─ 同步通知當班 Leader / 護理長
```

### 路徑 C：立即危險

```
立即危險（持凶器、嚴重肢體攻擊、生命威脅）
    │
    ├─ 直接撥打 110
    │
    ├─ 同步撤離現場人員
    │
    └─ 通知駐警隊 + 當班 Leader
```

> **哪些情境可直接報警（110）？**
> - 持刀、持棍棒等凶器
> - 已造成人員受傷
> - 挾持人質或封鎖出入口
> - 恐嚇炸彈或縱火

## 三、事件後續處理

暴力事件發生後，行政處理與證據保存至關重要。

### 3.1 衛生局滋擾事件通報

- **時機**：事件發生後由 Leader 或護理長儘速通報
- **方式**：依衛生局滋擾事件通報流程填報
- **注意**：逐項確認通報單內容是否皆已完成

### 3.2 院內異常事件通報單

- **時機**：事件發生後 24 小時內完成
- **填報者**：當班 Leader 或指定同仁
- **內容**：事件經過、處置過程、傷害評估、改善建議

### 3.3 病歷紀錄

- 在病患病歷中記載事件經過
- 客觀描述行為（避免主觀評斷用語）
- 記錄處置措施及後續追蹤計畫

### 3.4 監視畫面保存

- 事件發生後，立即通知相關單位調閱並保存監視器錄影
- 密錄器/手機錄影檔案統一交由 Leader 保管
- 保存期限應符合院內規定，作為後續法律程序之佐證

## 四、人員關懷

暴力事件的受害者不只是身體上的傷害，心理衝擊往往更為持久。院方有責任提供完整的關懷支持。

### 4.1 事後關懷流程

| 時間 | 關懷動作 | 負責人 |
|------|---------|--------|
| 事件當日 | 確認同仁身心狀況，協助就醫及驗傷 | Leader / 護理長 |
| 事件後 1--3 日 | 主動關懷、詢問是否需要調整班次 | 護理長 |
| 事件後 1--2 週 | 追蹤心理狀態，評估是否需轉介心理諮詢 | 護理長 / 職安室 |
| 事件後 1 個月 | 團隊檢討會議（debrief），回顧處置流程 | 單位主管 |

### 4.2 心理諮詢資源

| 資源 | 說明 |
|------|------|
| **院內心理諮詢** | 本院員工可申請心理諮商服務 |
| **衛生局「安心服務」** | 衛生局提供之復原關懷服務，可由院方協助轉介 |
| **員工關懷計畫** | 院內員工關懷計畫（[院內連結](https://km.ntuh.gov.tw/km/readdocument.aspx?documentId=90347)） |

> **同仁別忘了照顧自己**：經歷暴力事件後出現失眠、焦慮、反覆回想等反應是正常的。不需要「撐過去」，主動尋求支持是專業的表現。

### 4.3 筆錄流程

需要做筆錄時，同仁常感到緊張與不安。以下是重要的流程說明：

- **地點**：可請管區在**門診**製作筆錄，不一定要去警局
- **時間**：若上班中不便，可與管區**另約時間**至警局做筆錄
- **陪同**：若自行前往警局感到害怕，可請**本院駐警陪同**前往
- **工時**：至警局做筆錄之時間**可予加時數**，不需使用個人假別

## 五、相關法規

### 5.1 醫療法第 24 條

任何人不得以強暴、脅迫、恐嚇或其他非法方式，妨礙醫療業務之執行。違者處三年以下有期徒刑，得併科新臺幣三十萬元以下罰金。

### 5.2 精神衛生法重點條文

| 條文 | 要點 |
|------|------|
| 第 3 條 | 精神疾病定義與分類 |
| 第 32 條 | 嚴重病人有傷害他人或自己之虞時，得予以緊急安置 |
| 第 41 條 | 強制住院/社區治療相關規範 |

> **實務提醒**：門診遇到疑似精神疾病患者出現暴力行為時，降階溝通仍是第一選擇。但若無法溝通且有立即危險，應依法啟動緊急處置流程。

### 5.3 刑法相關

- **第 277 條（傷害罪）**：故意傷害人之身體或健康者
- **第 304 條（強制罪）**：以強暴、脅迫使人行無義務之事或妨害人行使權利者
- **第 305 條（恐嚇罪）**：以加害生命、身體、自由、名譽、財產之事恐嚇他人者

## 通報升級快速檢核表

事件發生時，按順序確認：

- [ ] 是否出現四項升級指標之一？（言語威脅 / 持物 / 逼近 / 無法溝通）
- [ ] 已依決策樹啟動對應通報路徑
- [ ] 駐警到場前已開啟密錄器或手機錄影
- [ ] 事件後已完成：衛生局通報 + 院內異常事件通報單 + 病歷紀錄
- [ ] 已保存監視畫面
- [ ] 已關懷受影響同仁，告知心理諮詢資源
- [ ] 需做筆錄者已安排時間與陪同
