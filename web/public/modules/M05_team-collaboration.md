# M05 團隊合作與安全處置

> **學習目標**：學員能說明門診暴力事件中的團隊分工模型，正確執行 call-for-help 啟動流程，並運用安全站位原則保護自己與周圍人員。

---

## 一、團隊分工三角色模型

門診暴力事件中，單打獨鬥是最危險的處置方式。團隊協作的核心是「三角色制」——每位在場同仁快速辨識自己的角色，各司其職。

### 1. 主責溝通者（Primary Communicator）

- **職責**：負責與激動病人/家屬進行一對一溝通，運用「停聽同選」/ DEFUSE 技巧降階
- **原則**：現場只能有一位主責溝通者，避免多人同時說話造成病人混亂與更大激動
- **人選建議**：
  - 優先由當下與病人互動最多的同仁擔任（通常是該診間護理師）
  - 若該同仁本身是衝突對象，則由另一位同仁接手
  - 主管（leader / 護理長）到場後，可視情況決定是否接手

> **指定原則**：第一位到場的資深同仁以明確口語指定：「我來跟他說，你去打電話。」避免大家都想溝通、或大家都在旁觀。

### 2. 通報者（Alerter / Reporter）

- **職責**：負責啟動 call-for-help 流程，依序聯繫護理站、當班 leader、駐警隊
- **執行要點**：
  - 退至安全位置（例如護理站）撥打電話
  - 通報時簡述：地點、事件類型、病人狀態、是否需要駐警
  - 通報完畢後回到現場支援或協助引導其他病人

### 3. 現場秩序維護者（Scene Manager）

- **職責**：管理現場環境，保護其他病人與同仁安全
- **執行要點**：
  - 引導其他候診病人遠離衝突區域
  - 保護高風險族群（長者、孩童、行動不便者）
  - 避免群體圍觀（圍觀會升高衝突者的表演慾）
  - 清除潛在危險物品（候診椅、點滴架等）
  - 確保出口動線暢通

### 何時交接主責溝通者

在以下情境中，應主動交接主責溝通者角色：

- **降階嘗試 5 分鐘無明顯改善**：已運用停聽同選 / DEFUSE 但對方情緒未見緩和
- **溝通者本身是衝突觸發者**：病人針對你個人，換人可降低對立感
- **溝通者情緒受影響，無法維持冷靜**：自覺心跳加速、語氣變化、判斷力下降
- **對方要求「找主管來」**：尊重對方的需求，同時也是降溫的契機

**交接話術**：「○○，麻煩你來接手，我先去聯絡XX。」

> **原則**：交接應自然且專業，不是「承認失敗」，而是「團隊策略的一部分」。順暢的交接本身就是一種降階技巧。

> **與 M04 同理三句公式整合**：接手者第一句話最好用 [同理三句公式](/content/M04#section-6)：「**我注意到**您在跟我同事討論等候時間的事（事實）**，您是不是覺得**很焦慮（情緒）**？**您想要**有人幫您把流程問清楚（需求）**？**」— 展示「我有在聽」，避免病人被迫重頭講一次訴求。

## 二、門診 Call-for-Help 啟動流程

門診暴力事件的通報是一條「由近到遠」的鏈條：

```
診間同仁 → 護理站 → 當班 Leader → 駐警隊 → 管區派出所 → 110
```

### 啟動時機與層級

| 層級 | 啟動條件 | 聯繫對象 | 方式 |
|------|---------|---------|------|
| **第一層** | 病人情緒激動、言語衝突 | 護理站同仁 | 口頭呼叫或對講機 |
| **第二層** | 規勸無效、情勢升溫 | 當班 Leader / 護理長 | 分機電話 |
| **第三層** | 言語威脅或肢體動作 | 駐警隊 | 分機電話（東址/西址/兒醫各有專線） |
| **第四層** | 肢體暴力已發生 | 管區派出所 | 由駐警主動聯繫 |
| **第五層** | 立即危險、持凶器 | 110 | 直接撥打 |

> **重要提醒**：言語暴力階段，駐警到場後會先以規勸方式處理；由當班 leader 到現場評估後，再告知駐警是否需聯繫管區。肢體暴力則由駐警立即主動聯絡管區到院。

### 通報話術範本

> 「這裡是 XX 科診間，有一位病人（/家屬）情緒激動，目前有言語威脅（/拍打桌面/逼近同仁），請駐警前來支援。地點在 X 樓 X 診。」

## 三、駐警到場處理流程

駐警到場後的標準處置鏈：

| 步驟 | 動作 | 注意事項 |
|------|------|---------|
| 1 | 到場前即開啟密錄器（或手機錄影） | 留存證據，全程錄影 |
| 2 | 評估現場安全，確認威脅程度 | 與護理人員交接情況 |
| 3 | 規勸當事人冷靜 | 言語暴力先規勸 |
| 4 | 聯絡管區派出所 | 肢體暴力立即聯繫；言語暴力由 leader 評估後決定 |
| 5 | 管區員警到院 | 留下當事人資料、製作紀錄 |
| 6 | 若管區未積極處理 | 告知當班 Leader 及護理長 |

> **特別注意**：若現場管區員警欲請當事人私下了事，請務必告知當班 Leader 及護理長處理，不可自行接受私了。

## 四、安全站位與物理處置（團隊版）

### 基本站位原則（個人）

- 保持 1--2 公尺安全距離
- 側身約 45 度面對病人
- 靠近出口，確保退路暢通
- 不被逼至角落或死角
- 能坐就不要站（降低威脅感），但需確保可迅速起身

### 團隊站位策略（三人以上）

當現場有三位以上同仁時，建議採取以下站位配置：

```
        [出口]
          |
    秩序維護者 ←— 管理其他病人動線
          |
  主責溝通者 ←——→ 病人（1-2m）
          |
     通報者 ←— 退至護理站通報
```

**要點：**

1. **主責溝通者**：面對病人，側身 45 度，距離 1--2 公尺
2. **秩序維護者**：位於主責溝通者與出口之間，負責引導其他病人離開
3. **通報者**：退至安全位置（護理站），完成通報後可回到現場支援
4. 所有人保持出口動線暢通，**絕對不要圍堵病人**（圍堵會讓病人感到被困，升高攻擊風險）

### 引導其他病人遠離

- 秩序維護者以平穩語氣引導：「不好意思，麻煩往這邊移動一下，謝謝。」
- 優先疏散高風險族群：輪椅長者、帶著孩童的家長、行動不便者
- 避免使用「危險」等字眼，以免引發恐慌

## 五、指定主責溝通者的原則

多人同時與激動病人對話是降階失敗最常見的原因之一。以下是指定主責溝通者的建議：

| 情境 | 主責溝通者人選 |
|------|------------|
| 病人針對特定護理師抱怨 | 由**另一位**同仁接手（換人可降溫） |
| 病人對流程不滿，尚未針對個人 | 由**當下互動最多**的同仁繼續 |
| 主管（Leader）到場 | 由主管**評估後決定**是否接手 |
| 駐警到場 | 駐警負責安全，**溝通仍由護理師主導**（除非已升級為治安事件） |

> **口語指定範例**：
> - 「小美，你繼續跟王先生說，我去打電話。」
> - 「學姊，這邊我來處理，麻煩你幫忙帶其他病人到那邊等。」

## 六、病友疏散流程

根據 4/7、4/9 前測會議共同發現：現場支援人員雖多，但缺乏明確分工與行動指引，容易形成圍觀。病友疏散是「現場秩序維護者」最核心的任務之一。

### 疏散優先順序

衝突現場的疏散並非亂中撤離，而應依風險等級分批處理：

| 順序 | 對象 | 原因 |
|:---:|------|------|
| **第一優先** | 孩童、長者、行動不便者 | 反應慢、易受傷、無自保能力 |
| **第二優先** | 孕婦、需陪同就醫者 | 情緒易波動、移動需額外協助 |
| **第三優先** | 其他候診民眾 | 人數最多，應分批有序引導 |

### 疏散動線規劃

- **遠離衝突現場**：引導方向應與激躁者所在位置相反
- **避免通過診間前方**：避開激躁者的視線與攻擊範圍
- **預設集合點**：如候診區後方、鄰近診區等候空間，讓病友有明確目標
- **動線寬敞**：單一方向流動，避免對撞或回流

### 專責人員與話術

疏散工作由 M05 三角色中的**現場秩序維護者（Scene Manager）**執行。話術要領：

- **語氣平穩**：「不好意思，麻煩您先到候診區後方，謝謝。」
- **避免使用「危險」「暴力」等字眼**：以免引發恐慌、群體推擠
- **給予明確指令**：告知**要去哪裡**（而非只叫對方「離開」）
- **重複並堅定**：若民眾猶豫或想圍觀，重複指令並引導方向

### 避免圍觀的關鍵動作

圍觀會升高激躁者的「表演慾」與對峙感，必須主動破除：

1. **明確指令**：「請您先到候診區後方」比「請大家散開」更有效
2. **身體引導**：以手勢、視線明確指出方向
3. **轉移注意**：「您的門診還在 X 診，請先到那邊等候」
4. **階段性清場**：先清離直接視線範圍，再清離整個區域

> **4/7 會議觀察**：前測發現現場人員「雖多但缺分工」，容易形成圍觀。疏散不是讓所有人離開，而是讓**該離開的人離開、該留下的人各司其職**。

## 七、事件中的密錄器使用

密錄器（或手機錄影）是保護同仁權益的重要工具：

- **誰要開啟**：駐警到場前即應開啟隨身密錄器；護理同仁可使用手機錄影
- **錄影目的**：留存證據，作為後續通報、筆錄、法律程序的依據
- **注意事項**：
  - 錄影時不要刻意靠近或激怒對方
  - 保持穩定拍攝，盡量涵蓋現場全景
  - 錄影檔案應交由 leader 統一保管，不得外流或上傳社群媒體

### 密錄器啟動時機

- **啟動時機**：當出現任何升級指標時即應開始錄影（參見 [M06 四項升級指標](M06_reporting-regulations.md)：言語威脅、持物揮舞、逼近醫護、無法溝通）
- **錄影目的是證據保全，不是挑釁**：保持客觀、穩定，不要用錄影行為刺激對方
- **告知錄影**：若情況允許，主動告知：「為了保障雙方權益，我們會進行錄影。」
- **無密錄器時的替代方案**：
  - 使用手機錄音（至少保留音訊證據）
  - 確認該區域是否有監視器覆蓋
  - 事後立即書面記錄事件經過

### 支援請求標準內容（Pre-SBAR）

根據 4/7、4/9 前測會議共同發現：**駐警到場前資訊不足，無法攜帶適當裝備**。因此，標準 SBAR（較完整，到場後使用）之前，還需要一個**極短版 Pre-SBAR**，用於暗語呼救當下。

**Pre-SBAR 格式**（到場前速報版，5--10 秒可說完）：

```
○○診 + 緊急度 + 是否持械 + 現場人數 + 四大樣態標籤
```

**範例：**

- 「內科 13 診 + 緊急 + 未持械 + 醫護 2 人、病人 1 人 + **LOC 主導**（情緒衝動）」
- 「精神科 5 診 + 非常緊急 + 疑似持刀 + 門已反鎖 + 醫護 2 人 + **LOR 明顯**（幻聽+多疑）」

> **為什麼加四大樣態標籤？** 駐警／支援同仁到場前就知道「病人主要處於 LOR（失去現實感）」時，會自動調整**不正面對視、不突然動作、保持距離**，比臨場判斷快數秒。標籤源自 [M02 四大樣態](/content/M02#section-2)。

### Pre-SBAR vs SBAR 對照

| 面向 | Pre-SBAR（到場前） | SBAR（到場後） |
|------|-------------------|----------------|
| **使用時機** | 暗語呼救當下、駐警出發前 | 駐警抵達現場交接時 |
| **長度** | 5--10 秒 | 30 秒以內 |
| **目的** | 讓駐警決定攜帶何種裝備 | 讓駐警掌握全貌以便介入 |
| **必備欄位** | 地點+緊急度+是否持械+人數 | S+B+A+R 四項 |
| **通報管道** | 分機、對講機、暗語 | 面對面口語交接 |

> **4/7、4/9 會議結論**：「通報時未具備標準化交班內容」「駐警接獲通報資訊不足，無法預先準備適當防護裝備」。Pre-SBAR 的核心價值在於**讓駐警出發時就知道該帶什麼（盾牌？Master Key？束帶？）**，而非到場才臨機應變。

### 駐警到場時的簡報內容（SBAR）

駐警到場後，主責溝通者或通報者應以 **SBAR 格式**進行簡潔簡報，讓駐警快速掌握情況：

| SBAR | 中文意涵 | 簡報內容 |
|:----:|---------|---------|
| **S** | Situation（什麼事） | 事件類型與當前狀態 |
| **B** | Background（誘因） | 導致衝突的原因 |
| **A** | Assessment（目前狀態） | 對方目前的情緒與行為表現 |
| **R** | Recommendation（需要什麼協助） | 你希望駐警做什麼 |

> **簡報範例**：「這位先生因為等候時間過長情緒激動（S+B），目前有言語威脅但沒有肢體動作（A），請協助維持現場秩序（R）。」

簡報要點：
- 控制在 **30 秒以內**
- 先講結論（目前安不安全），再補充背景
- 明確說出你需要什麼協助

### 駐警裝備配置差異

4/9 前測會議（精神部）觀察：**駐警支援精神科時配有盾牌等裝備，一般門診尚未配置**。這項差異直接影響現場應變能力。

**裝備差異對照：**

| 裝備 / 支援面向 | 精神科門診 | 一般門診 | 差異影響 |
|--------------|----------|---------|---------|
| **盾牌** | 有配置 | 通常無配置 | 持械情境下防護力差異大 |
| **約束用具**（束帶、軟式約束） | 有配置 | 多無配置 | 需改化學約束或等急診支援 |
| **鎮定藥物 SOP** | 精神科醫師可立即處方 | 需聯繫精神科會診 | 反應時效差距可達數十分鐘 |
| **駐警受訓時數** | 較熟悉精神病人互動 | 一般警衛訓練為主 | 溝通與降階能力差異 |
| **Master Key** | 通常配置 | 視院區而定 | 反鎖事件反應速度 |

**差異原因：**

1. **風險預設差異**：精神科長期被視為高暴力風險科別，資源配置較完整
2. **歷史因素**：一般門診暴力升高是近年現象，裝備配置未跟上
3. **預算與空間**：盾牌、約束用具需儲存空間與定期檢核，全院配置成本高

**建議改善方向（4/9 會議結論）：**

1. **跨單位一致性原則**：建立全院基本裝備底標（至少 Pre-SBAR 能準備的項目）
2. **高風險診區加強**：內科、急診、腎臟科等高衝突熱點診區，比照精神科配置
3. **機動支援模式**：Pre-SBAR 若提示「持械」或「反鎖」，駐警即從中央裝備站攜帶盾牌/Master Key 前往
4. **駐警分級受訓**：依支援科別不同，提供對應情境的進階訓練

> **實務意涵**：裝備差異不是「誰比較重要」的問題，而是「風險特性不同需不同工具」。跨單位一致性的目標不是齊頭式配置，而是**確保每個診區在自身風險範圍內都有足夠資源**。

## 團隊合作快速檢核表

在事件發生時，快速確認以下事項：

- [ ] 已指定主責溝通者（只有一人在跟病人說話）
- [ ] 已有人負責通報（護理站 → leader → 駐警）
- [ ] 已有人引導其他病人遠離
- [ ] 所有人保持安全距離與站位
- [ ] 出口動線暢通，無人被逼至角落
- [ ] 密錄器或手機已開啟錄影
- [ ] Pre-SBAR 已含四大樣態標籤（LOR/LOC/LOH/LOP）
- [ ] 若將進入**伐兵**（強制力介入），對照 [M07 伐兵 checklist](/content/M07#zero)（安全壓制需 5 人、裝備、法律依據）
