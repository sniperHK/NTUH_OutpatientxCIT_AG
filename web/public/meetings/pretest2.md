# 門診暴力防範情境模擬前測會議紀錄（第 2 場）

> **時間**：2026-04-09（三）
> **地點**：精神部門診診間
> **參與人員**：精神科、口醫部醫師、門診護理師、護理長、門診部同仁、安衛室同仁、駐警隊同仁等

## 暴力防範情境模擬前測討論

## 一、前測現況與問題分析

### （一）人員應變

1. 面對持刀械病人，是否適合由單一人員近距離接觸，尚無明確指引
2. 有人員反應會直接制止或抓住病人手部，顯示缺乏安全優先之標準作法
3. 部分單位（如精神科）醫護以非語言方式（眼神）傳遞支援需求，未正式啟動支援機制
4. 第二支援角色介入，有助於情緒緩和及問題再表述

### （二）溝通方式

1. 以「協助解決問題」及「提供第二意見」方式，具降低病人情緒之效果
2. 病人情緒主因多與費用相關，若能即時回應需求，有助降低衝突
3. 精神科反映警報裝置，響鈴明顯，更可能刺激病人，傾向避免使用
4. 醫護與駐警間缺乏標準化交班內容，影響應變效率

### （三）設備與安全支援

1. 駐警到場前未接獲完整資訊（如是否持械、是否反鎖門），亦未攜帶必要裝備（如 Master Key）
2. 精神科約束設備老舊，影響安全性與使用效益
3. 緊急壓扣鈴於診間內會發出聲響，易刺激病人，建議調整為無聲通報
4. 精神科具備鎮定藥物處置流程，惟各門診單位作法未一致
5. 駐警支援精神科時配有盾牌等裝備，一般門診尚未配置

### （四）環境與動線應變

1. 病人反鎖門時，駐警無法即時進入，僅能破門，應變彈性不足
2. 診間空間設計未充分考量緊急撤離需求（如診桌下方可供暫時避難）
3. 缺乏單門與雙門診間之標準撤離策略
4. 現場支援人員雖多，惟缺乏明確分工與行動指引

## 二、結論與後續建議

1. 建立持械情境之標準應對原則（避免單人近距離接觸）
2. 制定醫護與駐警之標準交班內容（如是否持械、是否反鎖門、現場人數及情緒等）
3. 建立低刺激之通報機制（如無聲警報系統）
4. 明確規範現場人員分工及第二支援角色任務
5. 規劃診間撤離動線（單門／雙門）及應變策略
6. 強化駐警裝備配置與應變能力
7. 檢視精神科與一般門診處置差異，建立跨單位一致性原則
