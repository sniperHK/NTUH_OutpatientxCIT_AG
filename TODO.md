  - 負責人：雯霞團隊

  - 方式：先射箭再畫靶，邊演練邊修正

  - 負責人：劉醫師
  - 背景：急診部專家，有消防局/警察局 CIT 推廣經驗
  - 目標：規劃半天～一天的工作坊內容

- [ ] **諮詢謝主任關於跨單位訓練可行性**
  - 負責人：雯霞

## 優先順序 2：課程規劃

- [ ] **確認工作坊參加對象**
  - 負責人：雯霞團隊
  - 第一階段：核心成員（後續擔任評核老師）
  - 需納入：門診部安全室人員

- [ ] **確認工作坊時間安排**
  - 負責人：雯霞團隊
  - 目標時長：半天～一天（非兩天完整版）
  - 需配合：核心成員可調動的時間

- [ ] **調整演練評核表**
  - 負責人：劉醫師
  - 改進方向：
    - 行為指標要更明確具體
    - 通報對象、電話要寫清楚
    - 配合 CIT 與 SOP 內容調整
  - 評核老師：需 2 位（避免主觀或漏看細節）

---

## 優先順序 3：評估工具

- [ ] **準備前後測問卷**
  - 負責人：劉醫師提供
  - 內容：
    - 自信心問卷（自我效能評估）
    - 知識測驗（情境考題：安全距離、溝通方式等）
  - 前測可含問答題，同時作為現況調查

- [ ] **評估標準化病人（SP）預算與排程**
  - 負責人：雯霞團隊
  - 費用：約 $600/半天
  - 資源：精神中心有 SP 資源
